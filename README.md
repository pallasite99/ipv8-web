It showcases a modern visual infrastructure designed to bypass legacy BGP-based routing tables in favor of context-aware neural resolution, post-quantum cryptography, and space-sea-ground mesh networks.

## ✨ Features Overview

### 🌍 1. 3D Interactive Network Globe
*   **Vector Node Projections:** Real-time projected coordinate transformations on a custom 2D-context 3D sphere.
*   **Zero Glitch Rotation:** Drag, orbit, and spin without rotation-reset or snapping bugs.
*   **HUD Telemetry & Logs:** Live telemetry streaming demonstrating active node-to-node orbital path optimization and zero-trust handshakes.

### ⚡ 2. Visual Architecture Simulator (Sandbox)
*   **BGP Congestion Simulation:** Inject artificial congestion bottlenecks to see how standard legacy packets (red) crawl, while IPv8 packets (cyan) automatically redirect around congestion nodes.
*   **Dynamic Metrics Panel:** Real-time graphs showing Latency (ms), Packet Loss Rate (%), Power Coefficient (W/GB), and Network Hops in simulated transit.
*   **No-Collision Key Rendering:** Powered by a clean, robust, and leak-free React state engine with custom ref-based unique identifier tracking.

### 💻 3. Developer Portal & Sandbox CLI
*   **Endpoint Simulator:** Run live sandboxed simulation requests for `POST /resolve` (AI route translation) and `POST /tunnel` (Post-quantum tunnel sessions).
*   **Multilingual Client SDK SDKs:** Real interactive code snippets in **TypeScript**, **Rust**, **Go**, and **Python**.
*   **Dynamic Run-Cancellation:** Select a new language tab or click an endpoint mid-simulation, and active processes are cancelled gracefully with no log leaks or overlapping outputs.

## 🏃 Getting Started Locally

To run the repository locally, follow these simple steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) (version 18+) installed.

### Installation
1. Clone your repository:
   ```bash
   git clone https://github.com/pallasite99/ipv8-web.git
   cd ipv8-web
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Spin up the local development server:
   ```bash
   npm run dev
   ```

4. Build the application for production:
   ```bash
   npm run build
   ```
