# 🔬 Phase 1: Semantic Packet Inspector (Deep Packet Analysis)

The **Semantic Packet Inspector** provides real-time observability into the cNAT (Context Network Address Translation) and semantic headers generated by the IPv8 protocol.

---

## 🎯 Value Proposition & Goals

In the current version, packets move across the screen as generic colored dots. While visually appealing, it remains an abstraction. Phase 1 introduces **Deep Packet Analysis (DPA)**, allowing developers to inspect exactly *what* is inside an IPv8 context-packet and how it contrasts with a legacy IPv4/IPv6 packet.

```
       [ Legacy Packet ]                            [ IPv8 Packet ]
┌──────────────────────────────┐            ┌──────────────────────────────┐
│ Src: 192.168.12.84           │   cNAT     │ Src-Hash: 0x8aef...3e2a      │
│ Dst: 104.244.42.1            │ ─────────> │ Neural Vector: [0.84, -0.12] │
│ Protocol: TCP                │            │ Crypto: ML-KEM-1024          │
│ Port: 443                    │            │ Semantic Intent: CDN_QUERY   │
└──────────────────────────────┘            └──────────────────────────────┘
```

---

## 🎨 Visual Component Design

### 1. The DPA Inspection Panel
*   **Layout**: A side-docking glassmorphic HUD panel (`backdrop-blur-md bg-slate-950/70 border-l border-cyan-500/30`) that animates onto the screen when a packet is clicked or when the "Inspect" tool is active.
*   **Dynamic Visualizer**: A visual representation of a binary bitstream decoding live, styled in deep green and cyan monospaced typography (`font-mono text-xs text-cyan-400`).
*   **Comparison Matrix**: A side-by-side comparison displaying:
    *   **Legacy IPv4/v6 header fields** (Source IP, Destination IP, TTL, Port, Protocol).
    *   **Speculative IPv8 headers** (Neural Coordinate Vector, Semantic Class, ML-KEM Session Key, Context Hash).

### 2. Interactive Packet Freeze & Hover
*   **Interaction**: Adding a "Pause/Step Simulation" toggle in the interactive sandbox container.
*   **Packet Clicking**: Packets flowing through the simulator gain hitboxes and hover animations. Clicking a packet freezes its position with a subtle glowing target ring, loading its real calculated properties directly into the DPA panel.

## ⚙️ Core State & Data Structure additions

We will define new structures in `src/types.ts` to support this data:

```typescript
export interface SemanticHeader {
  id: string;
  legacySrc: string;
  legacyDst: string;
  semanticSrc: string; // 512-bit hex coordinate hash
  neuralVector: number[]; // [x, y, z] spatial neural coordinate
  semanticIntent: 'API_DISPATCH' | 'MEDIA_STREAM' | 'LEDGER_SYNC' | 'QUANTUM_TUNNEL';
  cryptoSignature: string; // ML-KEM signature fragment
  latencyOffset: number; // calculated routing saving
}
```
